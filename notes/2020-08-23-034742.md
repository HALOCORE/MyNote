- Building LLVM with CMake

  - https://llvm.org/docs/CMake.html

  - 需要先安装最新的CMake

    - 可以源码安装或者.sh安装

    - .sh运行后只是做了解压

    - 源码安装

      ```
      先./bootstrap
      然后make
      接着sudo make install
      
      如果找不到openssl,
      sudo apt-get install libssl-dev
      ```
